# Arch Companion Philosophy

## Learn before doing

The application explains every action before executing it.

## Transparency

Commands are never hidden.

## Respect Arch Linux

Arch Companion follows Arch Linux best practices and encourages users to understand their system.

## Simplicity

Keep the code simple.

## Reusability

Reusable components are preferred over duplicated code.

## Documentation First

Architecture and design decisions are documented.
