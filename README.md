# Arch Companion

> **Learn Arch. Don't fight it.**

Arch Companion is an educational assistant for Arch Linux.

Its goal is not to hide Linux commands, but to help users understand them.

Every action is transparent, explained, and follows Arch Linux best practices.

---

# Philosophy

Arch Companion believes that users should learn how their system works instead of blindly clicking buttons.

Before executing important actions, the application explains:

- what command will be executed;
- why this command is used;
- what will happen;
- where to find the official documentation.

The Arch Wiki always remains the primary reference.

---

# Principles

- Explain before executing.
- Follow Arch Linux best practices.
- Never hide important commands.
- Prefer official tools whenever possible.
- Teach instead of replacing knowledge.

---

# Modes

## Learning Mode

Recommended for new Arch users.

Features:

- command explanations
- confirmations
- Arch Wiki links
- educational descriptions

---

## Expert Mode

Designed for experienced users.

Features:

- faster workflow
- fewer confirmations
- same safe operations
